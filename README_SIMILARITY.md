# 유사도 기반 방 배정 시스템

## 개요
이 시스템은 학생들의 특성(추위 민감도, 소음 민감도 등)을 기반으로 유사한 학생들끼리 같은 방에 배정하는 기능을 제공합니다.

## 특성 컬럼
엑셀 파일에 다음 컬럼을 추가하면 유사도 기반 배정이 자동으로 적용됩니다:

- **ColdSensitivity** (1~10): 추위 민감도
  - 1: 추위를 전혀 타지 않음
  - 10: 추위를 매우 많이 탐

- **NoiseSensitivity** (1~10): 소음 민감도
  - 1: 소음에 무관심
  - 10: 소음에 매우 민감

- **LightSensitivity** (1~10): 빛 민감도
  - 1: 빛에 무관심
  - 10: 빛에 매우 민감

- **Cleanliness** (1~10): 청결도 선호도
  - 1: 청결도 무관
  - 10: 매우 깔끔함을 선호

- **SocialLevel** (1~10): 사교성
  - 1: 내성적
  - 10: 매우 사교적

- **StudyTime** (1~10): 학습 시간 선호도
  - 1: 짧은 학습 시간
  - 10: 긴 학습 시간

- **SleepTime** (1~10): 수면 시간 선호도
  - 1: 늦게 자는 것을 선호
  - 10: 일찍 자는 것을 선호

## 테스트 데이터 생성

테스트용 엑셀 파일을 생성하려면:

```bash
pip install pandas openpyxl numpy
python create_test_data.py
```

이 명령어를 실행하면 `test_data.xlsx` 파일이 생성됩니다.

## 알고리즘 동작 방식

1. **기존 조건 우선**: 
   - 이전 룸메이트 회피 (Prev1, Prev2, Prev3)
   - 회피 학생 제외 (Avoid1, Avoid2)
   - 블랙리스트 조합 제외

2. **유사도 계산**:
   - 조건을 만족하는 후보 학생들 중에서
   - 방의 현재 구성원들과의 유사도를 계산
   - 유클리드 거리를 사용하여 특성 벡터 간 거리 계산
   - 거리가 가까울수록 유사도 점수가 높음

3. **배정 선택**:
   - 유사도 점수가 가장 높은 학생을 선택하여 배정

## 사용 방법

1. 엑셀 파일에 위의 특성 컬럼들을 추가하고 1~10 값 입력
2. GUI에서 파일 선택 후 배정 실행
3. 시스템이 자동으로 유사한 학생들끼리 배정

## 참고사항

- 특성 컬럼이 없어도 기존 방식으로 배정됩니다
- 일부 특성만 있어도 해당 특성만으로 유사도 계산
- NaN 값은 기본값(5.0)으로 처리됩니다
